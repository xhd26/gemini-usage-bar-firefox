# Privacy Policy for Gemini Usage Bar

**Last Updated:** June 28, 2026

Your privacy is extremely important to us. This Privacy Policy describes how the **Gemini Usage Bar** Firefox Extension handles information.

## 1. Information Collection and Use

- **No Personal Data Collection**: The extension does **not** collect, store, or transmit any personal information, browsing history, or user data.
- **Local Operation**: All scraping and processing of usage statistics are done entirely locally on your device within your browser session.
- **No Third-Party Communication**: The extension only makes same-origin network requests to the official Gemini usage page (`https://gemini.google.com/usage`). It does not send any data to external servers or third-party services.

## 2. Browser Storage

- The extension uses `browser.storage.local` solely to store your display preferences (e.g., whether the usage pill is hidden or visible). This data remains on your local machine and is never shared.

## 3. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted directly to this page in the repository.

## 4. Contact

If you have any questions or suggestions regarding this privacy policy, please open an issue in the public GitHub repository.
